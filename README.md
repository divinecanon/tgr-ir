# TGR-IR 目标、语法规范、世界公理 v1.0
将 TGR 文档体系中的高维哲学概念，翻译为一种稳定的、可计算的中间表示（IR）。

为确保 IR 的严谨性，任何推演必须遵守以下四条铁律：
规范一：阶层降维律（禁止概念跳跃）
内容：任何非一阶原语的高阶概念，必须且只能由低阶概念逐层推导得出。严禁直接调用高阶词汇作为逻辑前提。
必然性：一阶原语不可再分，引入未拆解的高阶词将破坏体系的纯粹性。
规范二：链路闭合律（禁止半截推演）
内容：每一次状态变化必须走完完整链路：T（张力）累积 -> G（生成）发生 -> R（轨迹）延展 -> P（投影）反馈。严禁中途切断链路。
必然性：原语间为严格因果驱动，链路不闭合等同于接受“无中生有”。
规范三：属性纯粹律（禁止隐喻实体化）
内容：描述状态改变时，必须且仅表述为 L（维度）或 D（矢量）的变化。严禁引入“能量、质量、容器”等自然语言隐喻。
必然性：隐喻自带未定义属性，会使推演从“逻辑运算”滑向“文学想象”。
规范四：主体归因律（禁止上帝视角）
内容：任何动作必须归因于具体的 P 主体。严禁使用“系统自动”、“环境导致”等无主语表达。
必然性：P 是唯一的主动作实体，允许无名主体将破坏系统的封闭性与可计算性。

在本 IR 体系中，没有预设的时间轴或空间网格。
存在本质：一切皆是 P 之间的相互作用。
构建逻辑：P 发射与接收 C；T 驱动 G；G 产生 R。
高阶映射：一切高阶概念（时间、空间、物质、意识），均是对这一 P 与 G 过程的分辨率降维与标记。
