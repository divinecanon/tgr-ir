# TGR-IR 核心规范 v1.0 
一、 总纲：目标与原则
1. 核心目标
将 TGR 文档体系中的高维哲学概念，翻译为一种稳定的、可计算的中间表示（IR）。
2. 基石原则
极简底座：IR 底层保持极简，底层只描述机制发生，一切概念均为 P 的结果。
自洽封闭：严禁引入任何外部隐喻或新增概念，必须用已有原语自洽表达。
二、 底层原语集
系统由七个核心原语构成，分为存在系、动力系与轨迹系。
1. 存在系原语
C (Constraint - 约束/状态)
定义：系统的当前存在状态。C 没有实体，直接且仅由 L 与 D 表示。
属性：一切状态皆是 C。
L (Dimension - 维度)
定义：C 的属性，表示尺度或层级（原 Scale）。
D (Vector - 矢量)
定义：C 的属性，表示指向或趋势。
2. 动力系原语
P (Projection - 投影)
定义：唯一的主动作。P 既是发射 C 的主体，也是接收 C 的主体，同时是 P 的动作本身。反推可知，C 是被 P 发射与接收的对象与状态。
机制：P 必须在 L 的梯度上进行双向循环（高 L 向低 L 发射，低 L 向高 L 反馈）。
T (Tension - 张力)
定义：不可兼容 C 的共存态。当两个 C 的 D 不一致时产生 T。T 是引发系统改变的唯一动力。
G (Generation - 生成)
定义：对 C 的改变。G 先发生，P 后标记。
结果分类：对 G 结果的区分 P 为三种状态——未知（先验的变化）、重复（低 G 变化）、创造（G 超过分辨率阈值的变化）。
3. 轨迹系原语
R (Recursion - 轨迹)
定义：延展的路径痕迹。G 发生后，沿着 D 一致性最高的 L-D 节点留下的不可逆历史记录。
特性：R 并非必然连续，仅在一些 L-D 中形成连续 R。R 不是节点，但可以被标记为节点（如时间、空间、因果节点）。
三、 动力学运行机制
1. R 延展机制
R 沿 L-D 空间延展，由 P 驱动的双向循环与一致性累积决定：
双向 P 循环：上一轮的 C_R 作为高 L 状态，P 作为发射主体向低 L 发生 P；在低 L 发生 G 后，P 作为接收主体反向向高 L 发生 P。
D 一致与 T 形成：当反向 P 回来的状态作为新的 C 时，若这些 C 的 D 与上一次标记的 R 更一致，T 就会增大。
R 延展：在上述 T 作用下，R 更容易沿着上一次标记的 R 在 L-D 上的节点延展。
2. P 的动力学法则
盲发机制：C 在 T 和 G 的驱动下发生状态改变。走势仅由当前 C 的 D 决定。
状态落点：G 产生新的 C，其状态由当前的 L 和 D 决定。
L 标记法则：通过比较新 C 与旧 C 的 L 差异，标记这次 P 的性质：
若 L 发生净坍缩 -> 标记为“L 坍缩 P”。
若 L 发生净膨胀 -> 标记为“L 膨胀 P”。
四、 框架使用规范（语法四律）
为确保 IR 的严谨性，任何推演必须遵守以下四条铁律：
规范一：阶层降维律（禁止概念跳跃）
内容：任何非一阶原语的高阶概念，必须且只能由低阶概念逐层推导得出。严禁直接调用高阶词汇作为逻辑前提。
必然性：一阶原语不可再分，引入未拆解的高阶词将破坏体系的纯粹性。
规范二：链路闭合律（禁止半截推演）
内容：每一次状态变化必须走完完整链路：T（张力）累积 -> G（生成）发生 -> R（轨迹）延展 -> P（投影）反馈。严禁中途切断链路。
必然性：原语间为严格因果驱动，链路不闭合等同于接受“无中生有”。
规范三：属性纯粹律（禁止隐喻实体化）
内容：描述状态改变时，必须且仅表述为 L（维度）或 D（矢量）的变化。严禁引入“能量、质量、容器”等自然语言隐喻。
必然性：隐喻自带未定义属性，会使推演从“逻辑运算”滑向“文学想象”。
规范四：主体归因律（禁止上帝视角）
内容：任何动作必须归因于具体的 P 主体。严禁使用“系统自动”、“环境导致”等无主语表达。
必然性：P 是唯一的主动作实体，允许无名主体将破坏系统的封闭性与可计算性。
五、 世界观公理
在本 IR 体系中，没有预设的时间轴或空间网格。
存在本质：一切皆是 P 之间的相互作用。
构建逻辑：P 发射与接收 C；T 驱动 G；G 产生 R。
高阶映射：一切高阶概念（时间、空间、物质、意识），均是对这一 P 与 G 过程的分辨率降维与标记。
