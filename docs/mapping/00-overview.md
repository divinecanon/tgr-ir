# mapping/00-overview.md

> 表达层说明与映射索引。来源：TGR-IR 核心规范 v6.4 附录B（原"二阶映射与延伸研究"）。

**映射原则（表达层）**：IR 仅定义原语。任何成熟的理论体系均可视为 IR 在不同 L' 上的表达。具体理论清单与验证进度记录于 `ir-mapping` 仓库。

**表达与生成分离说明**：IR区分生成层与表达层。语言、数学、物理以及其他成熟理论均属于表达层。表达层可以完整描述IR生成出的结构，但不能替代结构本身的生成过程。因此，同一个IR结构可以拥有多个不同的表达，而不同表达之间的等价性，不构成IR底层原语等价性的证明。

**涌现定位**：涌现不是原语，而是 R' 自循环稳定后的导出结构再标记。

**D' 偏转成本研究问题**：偏转的数学本质、偏转成本如何从带宽中导出、R' 稳定性如何量化为阻力系数。（工程案例见 `ir-mapping/signal-for-llm.md`）。

**快慢变量映射**：已迁移至 `ir-mapping/fast-slow.md`。

**几何映射探索（IGMP）状态说明**：微分几何映射工作（IGMP七阶段）目前定位为"借用工具箱"性质的语义映射（阶段b：结构相似性论证），尚未按第一阶段五层编译流水线（含生成层数标注、依赖单调性核验）正式验证，不构成"IR已与微分几何完成严格对应"的依据。其"联络"结构此前依赖的"惯性"一词已删除，仅保留已推导部分（D'偏转消耗带宽产生T'，故系统在无额外张力驱动时倾向沿当前方向延续测定），是否需要"惯性"这一物理学标签仍属未决，暂不结案；"流形"评级按一致标准应下调为"条件继承"，而非"完全继承"。详见 `ir-mapping/differential-geometry.md`。

**带宽降质预测（信息保真度形状假说）**：由DeepSeek推导给出，验证前须先确认待预测量的定义域、Modulator机制在无调制层系统中的适用性等前置问题未决，详见 `ir-mapping/bandwidth-degradation-verification.md`。
