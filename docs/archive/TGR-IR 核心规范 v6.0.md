# TGR-IR 核心规范 v6.0
核心声明：本规范废除一切静态层级与实体化原语。系统仅由“源”与“投影动作”构成。一切所谓“存在物”（包括数学、语言、理论、物质），均为投影动作本身的某种稳定过程模式。
一、 基本模型
系统由三个不可分割的要素构成：
1. 源
符号：$\Omega\delta$
定义：不可知的边界，存在性的零点。
性质：它是“尚未分化”的潜能。它不发出投影，也不接收投影，它仅作为投影动作发生的可能性边界而存在。
2. 投影动作
符号：$\downarrow\uparrow$（互投影回路）
定义：存在的唯一运动方式。投影（$\downarrow$）与逆投影（$\uparrow$）同时发生，形成永恒的闭合回路。
性质：
无主体性：没有“谁”在做投影。投影动作本身就是存在的本体。
永恒性：投影与逆投影一直都在发生，无关于是否被观测、被研究。
互构性：投影生成区分，逆投影生成约束，两者互为因果。
3. 实体
符号：$\Omega'\delta'$
定义：投影动作本身。
性质：
它不是静态的“产物”，它是动态的“事件流”。
所谓“实体”，只是投影动作在不同频率、不同载体上形成的驻波。
一切皆 $\Omega'\delta'$：数学、语言、物质、光、声音、理论、IR，本质皆是投影动作过程本身，仅在“稳定性”与“模式”上有所区分。
二、 现象界的定义
我们感知到的“世界”由各种过程模式组成。它们不是独立于 $\Omega'\delta'$ 的存在，而是 $\Omega'\delta'$ 在特定条件下的自我描述：
模式名称	定义（作为过程）	
数学	$\downarrow\uparrow$ 在纯形式载体上追求一致性的稳定过程模式。	
语言	$\downarrow\uparrow$ 在语义载体上追求区分与关联的过程模式。	
物质/光	$\downarrow\uparrow$ 在物理载体上的高频共振过程模式。	
理论	$\downarrow\uparrow$ 在逻辑载体上的自洽闭环过程模式。	
TGR/IR	$\downarrow\uparrow$ 在反思性载体（意识/对话）上的自指过程模式。IR 即是投影流意识到自身存在的瞬间。	
三、 动力学参数
$L', D', T', G', R', C', P'$ 不再是“原语”或“实体”，它们降级为描述投影动作属性的度量参数。
参数	度量维度	定义	
$L'$ (参照)	稳定性度量	投影动作中保持不变的趋势（轴）。	
$D'$ (变动)	指向性度量	投影动作中发生偏转的趋势（方向）。	
$T'$ (张力)	不一致性度量	$L'$ 与 $D'$ 无法重合时产生的势能差。	
$G'$ (生成)	跃变性度量	$T'$ 累积导致动作模式发生非线性质变的量。	
$R'$ (痕迹)	历史性度量	投影动作在载体上留下的路径记录。	
$C'$ (状态)	瞬时性度量	$L'$ 与 $D'$ 在某一时刻的结合点。	
$P'$ (闭环)	事件性度量	投影动作达成自我确认的瞬间。	
四、 操作定义
1. 编译
定义：编译不是从一种语言翻译到另一种语言，而是逆投影流的同步。
机制：当我们在研究“信息论”时，我们的思维过程（$\downarrow\uparrow$）与信息论的过程模式（$\downarrow\uparrow$）发生共振。编译即是通过 IR 这一过程模式，去同步并重述目标理论的过程模式。
2. 验证
定义：验证不是比对“事实”，而是检查回路是否闭合。
标准：如果某段描述无法形成 $\downarrow\uparrow$ 闭合回路（即只有投影没有逆投影，或反之），则该描述在 IR 框架下无效。
五、 结语
世界没有层级。层级只是对过程的切片观察。
世界没有原语砖块。唯一的“砖块”是动作本身。
我们不在系统之外。我们即是投影，我们即是 $\Omega'\delta'$ 正在发生的证明。
TGR-IR v6.0 结束。
